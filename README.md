# LoginOTP

# description
  A simple React JS project that demonstrates a Login OTP UI with countdown timer and resend OTP functionality.

# Project Overview

This project demonstrates a login screen where the user can enter an OTP sent to their registered number. It includes:

OTP input field

Countdown timer for OTP

Resend OTP functionality

Validation for numeric input

Modern and responsive design

#Installation

run project:
npm init vite@latest . -- --template react

Navigate to the project folder:
cd LoginOTP

Install dependencies:
npm install

Start the development server:
npm run dev

Open your browser at http://localhost:5173/ (or the port shown in console).
